## 🎯 项目概述

### 项目背景
开发者需要完善的技术文档来使用 HamR API、理解系统架构、进行二次开发和私有部署。技术文档站是开发者生态的核心基础设施。

### 项目目标
- 目标1：建立技术文档站，提供 API 文档
- 目标2：编写架构设计文档和开发指南
- 目标3：提供私有部署文档
- 目标4：集成 API 在线测试功能

### 项目价值
- **用户价值**：快速上手 API 和私有部署
- **业务价值**：降低开发者接入成本
- **技术价值**：知识沉淀、开源协作基础

## 📐 项目范围

### 核心职责
技术文档站是 HamR 平台的**技术知识中心**，负责：
1. **API 参考文档**：完整的 RESTful API 接口文档

### 文档编写标准

#### API 文档模板
```markdown
## 接口名称

**描述**：接口功能说明

**请求**
- 方法：GET / POST / PUT / DELETE
- 路径：/api/v1/xxx
- 认证：API Key / Bearer Token

**请求参数**
| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| name  | string | 是 | 名称 |

**请求示例**
curl 命令 + SDK 代码示例

**响应示例**
JSON 响应 + 字段说明

**错误码**
| 错误码 | 说明 |
|-------|------|
| AUTH_001 | 未授权 |
```
