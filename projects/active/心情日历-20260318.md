## 🎯 项目概述

### 项目背景
现代人生活节奏快，情绪压力管理需求强烈，但缺乏简单直观的日常情绪记录工具。通过可视化的日历形式追踪每日心情，帮助用户了解自身情绪规律，实现轻量级的情绪和压力管理。

### 项目目标
- 目标1：提供极简的每日心情记录入口（点击日期即可选择心情）
- 目标2：在日历上直观呈现历史心情数据（颜色/emoji 标注）
- 目标3：生成心情曲线图，展示情绪趋势和规律

### 项目价值
- **用户价值**：帮助用户轻松记录每日情绪，发现情绪规律，管理心理压力
- **业务价值**：HamR 生态中的健康管理类应用，丰富产品矩阵
- **技术价值**：Vibe Coding 模式验证，探索 AI 辅助快速开发流程

## 🛠 技术方案

### 技术栈
- **前端框架**：React + TypeScript（Vibe Coding 模式）
- **UI 组件**：Ant Design（antd v5）+ dayjs
- **图表库**：自定义心情曲线图组件
- **数据存储**：localStorage（本地持久化）
- **构建工具**：Vite
- **部署**：Vercel
- **开发模式**：Vibe Coding（AI 辅助开发）

### 核心功能设计

#### 心情类型定义
| 心情 | Emoji | 颜色 | 分值 |
|-----|-------|------|------|
| 非常开心 | 😄 | #FFD700（金黄） | 5 |
| 开心 | 😊 | #90EE90（浅绿） | 4 |
| 平静 | 😐 | #87CEEB（天蓝） | 3 |
| 难过 | 😔 | #DDA0DD（薰衣草） | 2 |
| 很糟糕 | 😢 | #FF6B6B（珊瑚红） | 1 |

#### 页面布局
- **顶部**：月份导航（上/下月切换）+ 心情图例
- **中部**：月历视图（每格显示日期+心情emoji）
- **底部**：心情曲线图（近30天趋势）

#### 交互流程
1. 用户点击日历中的某一天
2. 弹出心情选择器（5种心情选项）
3. 选择后日历格子显示对应 emoji 和背景色
4. 曲线图实时更新

### 数据结构
```typescript
interface MoodEntry {
  date: string;        // 格式: YYYY-MM-DD
  mood: MoodType;      // 心情类型
  score: number;       // 情绪分值 1-5
  note?: string;       // 可选备注（后续扩展）
  createdAt: string;   // 记录时间
}

type MoodType = 'great' | 'good' | 'neutral' | 'sad' | 'terrible';
```
