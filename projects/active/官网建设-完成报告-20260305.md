## 📦 部署建议

### 推荐平台（零成本）

1. **Vercel** (推荐)
   - 优势：自动部署、预览环境、全球 CDN、零配置
   - 部署：连接 GitHub 仓库，自动构建
   - 域名：hamr.store → Vercel 项目
   
2. **Cloudflare Pages**
   - 优势：无限带宽、全球 CDN、免费 SSL
   - 部署：Git 集成，自动构建
   
3. **Netlify**
   - 优势：简单易用、表单处理、边缘函数

### 部署步骤（Vercel）

```bash
# 1. 安装 Vercel CLI
npm i -g vercel

# 2. 登录 Vercel
vercel login

# 3. 部署项目
cd repos/hamr-website
vercel

# 4. 绑定域名
vercel domains add hamr.store
```

### 环境变量（无需配置）
当前版本为纯静态站点，无需后端 API，不需要配置环境变量。
