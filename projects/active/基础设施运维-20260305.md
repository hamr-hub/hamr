## 🎯 项目概述

### 项目背景
HamR 平台的基础设施是所有服务的底层支撑，包括 DNS 解析、CDN 加速、SSL 证书、服务器管理、监控告警、数据备份等，需要持续维护和优化。

### 项目目标
- 目标1：完成 Phase 1 子域名的 DNS 配置和 SSL 证书
- 目标2：搭建监控告警体系（Prometheus + Grafana）
- 目标3：建立数据备份与恢复机制
- 目标4：制定应急响应预案
- 目标5：确保 99.9% 可用性

### 项目价值
- **用户价值**：稳定可靠的服务体验
- **业务价值**：保障业务连续性
- **技术价值**：运维体系建设、最佳实践积累

## 📐 项目范围

### 核心职责
基础设施运维是 HamR 平台的**底层支撑系统**，负责：
1. **DNS 管理**：所有域名的 DNS 解析配置和监控
2. **网络安全**：CDN 加速、WAF 防护、DDoS 防御
3. **证书管理**：SSL 证书的申请、续期、监控
4. **服务器运维**：服务器部署、配置、安全加固
5. **监控告警**：内部运维监控体系（区别于面向用户的状态页）
6. **数据安全**：备份策略执行、恢复演练、灾备方案

### 包含的功能

#### 1. DNS 管理
- [ ] **域名解析配置**
  - hamr.store 及所有子域名解析
  - hamr.top 及所有子域名解析
  - A/AAAA/CNAME 记录管理
  - MX 记录（邮件服务）
  - TXT 记录（SPF/DKIM/DMARC 邮件安全）
- [ ] **DNS 安全**
  - DNSSEC 启用和维护
  - DNS 劫持检测（定期比对解析结果）
  - DNS 变更审计日志
- [ ] **DNS 监控**
  - 解析延迟监控
  - 解析成功率监控
  - 异常解析告警

#### 2. CDN 与网络加速
- [ ] **Cloudflare CDN 配置**
  - 缓存策略（静态资源缓存 30 天，API 不缓存）
  - 页面规则（Page Rules）
  - 速率限制规则
  - 边缘计算（Workers - 按需）
- [ ] **性能优化**
  - Brotli 压缩启用
  - HTTP/2 + HTTP/3 启用
  - 图片优化（自动 WebP/AVIF）
  - 懒加载配置
- [ ] **CDN 监控**
  - 带宽使用统计
  - 缓存命中率
  - 请求量统计（按地区）

#### 3. SSL 证书管理
- [ ] **证书配置**
  - Let's Encrypt 通配符证书（*.hamr.store、*.hamr.top）
  - Certbot 自动申请与续期
  - 备用证书方案（Cloudflare Universal SSL）
- [ ] **证书监控**
  - 证书过期预警（提前 30 天/7 天/1 天）
  - 证书链完整性检查
  - HSTS 配置验证
- [ ] **证书轮换**
  - 自动续期脚本
  - 续期失败告警
  - 手动轮换流程

#### 4. 服务器管理
- [ ] **服务器配置**
  - 操作系统标准化（Ubuntu 22.04 LTS）
  - 初始化脚本（用户创建、SSH 配置、防火墙）
  - 软件安装标准化（Docker、Nginx、Node.js、Go）
- [ ] **安全加固**
  - SSH 密钥认证（禁用密码登录）
  - 防火墙规则（仅开放必要端口）
  - fail2ban 配置（防暴力破解）
  - 自动安全更新（unattended-upgrades）
  - 最小权限原则（非 root 用户运行服务）
- [ ] **资源管理**
  - 服务器资源清单（CPU/内存/磁盘/带宽）
  - 资源使用率监控
  - 容量规划和扩容预案
  - 成本优化

#### 5. 内部监控告警体系
- [ ] **指标采集（Prometheus）**
  - 系统指标（CPU、内存、磁盘、网络）
  - 应用指标（QPS、响应时间、错误率）
  - 数据库指标（连接数、查询延迟、缓存命中率）
  - 自定义业务指标（注册数、活跃用户数）
- [ ] **可视化（Grafana）**
  - 系统总览 Dashboard
  - 各服务独立 Dashboard
  - 数据库 Dashboard
  - 网络流量 Dashboard
  - 自定义告警规则面板
- [ ] **告警通知**
  - 告警级别（Critical / Warning / Info）
  - 通知渠道（邮件 + 钉钉/飞书 Webhook）
  - 告警静默和抑制规则
  - 告警升级（30 分钟未确认 → 升级通知）
- [ ] **告警规则示例**
  | 告警名称 | 条件 | 级别 |
  |---------|------|------|
  | CPU 高负载 | CPU > 85% 持续 5 分钟 | Warning |
  | 内存不足 | 可用内存 < 10% | Critical |
  | 磁盘空间 | 磁盘使用 > 90% | Critical |
  | 服务宕机 | 健康检查连续 3 次失败 | Critical |
  | 响应延迟 | P99 > 3 秒持续 10 分钟 | Warning |
  | 错误率高 | 5xx 错误率 > 5% | Critical |
  | 证书即将过期 | SSL 证书 < 7 天 | Warning |
  | 数据库连接池 | 连接使用率 > 80% | Warning |

#### 6. 日志管理（ELK Stack）
- [ ] **日志收集**
  - 应用日志（JSON 结构化格式）
  - 访问日志（Nginx access log）
  - 错误日志（Nginx error log）
  - 系统日志（syslog）
  - 安全日志（auth.log）
- [ ] **日志分析**
  - Kibana 可视化分析
  - 日志搜索和过滤
  - 日志模式识别
  - 异常日志告警
- [ ] **日志保留**
  - 热数据（7 天，SSD）
  - 温数据（30 天，HDD）
  - 冷数据（90 天，归档存储）
  - 超过 90 天自动删除

#### 7. 数据备份与灾备
- [ ] **备份策略**
  - **PostgreSQL 数据库**
    - 每日全量备份（pg_basebackup）
    - WAL 日志持续归档（PITR 支持）
    - 备份加密（GPG）
  - **Redis 数据**
    - 每日 RDB 快照
    - AOF 日志
  - **对象存储（MinIO）**
    - 每日增量备份（rsync）
    - 每周全量备份
  - **配置文件**
    - Git 版本控制
    - 每次变更自动备份
- [ ] **异地备份**
  - 备份文件上传到异地对象存储（S3 兼容）
  - 加密传输（TLS）+ 加密存储（AES-256）
  - 每周完整性校验（MD5/SHA256）
- [ ] **恢复演练**
  - 每月执行一次恢复演练
  - 恢复时间记录（RTO 目标：< 1 小时）
  - 数据完整性验证（RPO 目标：< 24 小时）
  - 演练结果记录和改进

#### 8. 安全防护
- [ ] **网络安全**
  - Cloudflare WAF 规则配置
  - DDoS 防护（Cloudflare 自动缓解）
  - IP 黑名单管理
  - 地域访问限制（按需）
- [ ] **应用安全**
  - 依赖漏洞扫描（定期）
  - 安全补丁更新
  - 容器镜像安全扫描
  - 密钥和敏感信息管理（环境变量 + Secret Manager）
- [ ] **安全审计**
  - 每季度安全审计
  - 渗透测试（每半年）
  - 安全事件记录和分析
  - 合规性检查

#### 9. 应急响应
- [ ] **应急预案**
  - 服务宕机响应流程
  - 数据泄露响应流程
  - DDoS 攻击响应流程
  - 域名劫持响应流程
- [ ] **响应流程**
  ```
  告警触发
    → 值班人员收到通知（< 5 分钟）
      → 初步评估影响范围（< 15 分钟）

            → 验证恢复
              → 发布故障公告（status.hamr.top）
                → 事后复盘（RCA）
  ```

  - 交接规范

## 🛠 技术方案

### 技术栈
- **DNS**：Cloudflare DNS
- **CDN**：Cloudflare CDN
- **SSL**：Let's Encrypt + Certbot
- **监控**：Prometheus + Grafana + UptimeRobot
- **日志**：ELK Stack（Elasticsearch + Logstash + Kibana）
- **备份**：rsync + Cron
- **安全**：Cloudflare WAF + Fail2ban

### 架构设计
```
域名规划（55个子域名）
  ↓
Cloudflare DNS 解析
  ↓
Cloudflare CDN 加速
  ↓
服务器集群
├─ Web 服务器（Nginx）
├─ 应用服务器（多个微服务）
├─ 数据库集群（PostgreSQL + MongoDB + Redis）
└─ 对象存储（MinIO）
  ↓
监控告警体系
├─ Prometheus（指标采集）
├─ Grafana（可视化）
├─ UptimeRobot（外部监控）
└─ 告警通知（邮件/钉钉）
  ↓
数据备份
├─ 自动备份（每日）
├─ 异地备份（每周）
└─ 恢复演练（每月）
```

### 技术依赖
- Cloudflare 服务
- Let's Encrypt 证书服务
