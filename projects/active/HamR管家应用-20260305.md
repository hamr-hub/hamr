## 🎯 项目概述

### 项目背景
HamR 管家是整个平台的核心应用，负责存储和管理家庭的人、时、事、物、境五维数据，辅助收集家族信息，评估家庭幸福感，保存大事记。这是产品价值的核心体现。

### 项目目标
- 目标1：实现五维管理基础功能（人员、时间、事务、物品、环境）
- 目标2：开发家庭数据看板，展示家庭状态
- 目标3：实现大事记功能，记录家庭重要时刻
- 目标4：支持数据导入/导出，保证数据主权
- 目标5：确保数据安全，支持端到端加密

### 项目价值
- **用户价值**：系统化管理家庭事务，提升家庭幸福感
- **业务价值**：核心产品功能，验证产品价值
- **技术价值**：五维数据模型实践，为 JiaBu 决策提供数据基础

## 📐 项目范围

### 核心职责
HamR 管家是平台的**核心数据管理应用**，负责：
1. **五维数据采集与存储**：人、时、事、物、境全方位数据管理
2. **家庭数据可视化**：数据看板、统计分析、趋势展示
3. **数据主权保障**：数据导入/导出、本地存储、端到端加密
4. **为 JiaBu 提供数据基础**：结构化数据供决策系统分析

### 包含的功能

#### 1. 人员管理（People）- 家庭成员维度
- [ ] **成员档案**
  - 基本信息（姓名、性别、生日、照片）
  - 联系方式（电话、邮箱、地址）
  - 健康档案（血型、过敏史、病史）
  - 教育背景、职业信息
- [ ] **关系图谱**
  - 家庭成员关系可视化（树状图/网状图）
  - 关系类型（父母、子女、配偶、兄弟姐妹）
  - 关系强度标记（亲密度）
- [ ] **重要日期**
  - 生日提醒（提前 7 天/1 天）
  - 纪念日管理（结婚纪念日、相识纪念日）
  - 自定义重要日期
- [ ] **成员动态**
  - 成员活动记录
  - 成长里程碑（孩子成长记录）
  - 照片墙（时间线展示）

#### 2. 时间管理（Time）- 时间维度
- [ ] **家庭日历**
  - 共享日历（全家可见）
  - 个人日历（仅自己可见）
  - 日程分类（工作、学习、娱乐、家务）
  - 日历视图（日/周/月/年）
- [ ] **日程管理**
  - 创建日程（标题、时间、地点、参与人）
  - 重复日程（每天/每周/每月/自定义）
  - 日程提醒（提前 15 分钟/1 小时/1 天）
  - 日程冲突检测
- [ ] **时间统计**
  - 时间分配分析（工作/家庭/个人时间占比）
  - 家庭活动频率统计
  - 时间利用效率评估

#### 3. 事务管理（Event）- 事件维度
- [ ] **待办清单**
  - 创建任务（标题、描述、截止日期、优先级）
  - 任务分配（指定负责人）
  - 任务状态（待办/进行中/已完成）
  - 子任务拆解
  - 任务标签分类
- [ ] **家庭大事记**
  - 重要事件记录（搬家、换工作、孩子入学）
  - 事件时间线展示
  - 事件照片/附件
  - 事件回顾与总结
- [ ] **家庭活动**
  - 活动计划（旅行、聚会、运动）
  - 活动预算管理
  - 活动参与人员
  - 活动照片集
- [ ] **决策记录**
  - 家庭重大决策记录（买房、换车、教育选择）
  - 决策过程记录
  - 决策结果追踪

#### 4. 物品管理（Thing）- 物品维度
- [ ] **家庭资产**
  - 资产登记（房产、车辆、电子设备、家具）
  - 资产价值记录
  - 购买日期、保修期
  - 资产照片/发票
- [ ] **物品分类**
  - 自定义分类（电器、家具、衣物、书籍、玩具）
  - 物品数量统计
  - 物品状态（正常/维修/报废）
- [ ] **位置管理**
  - 物品存放位置（客厅、卧室、储藏室）
  - 位置地图标记
  - 快速查找物品
- [ ] **维护记录**
  - 维修保养记录
  - 更换配件记录
  - 维护提醒（定期保养）
- [ ] **消耗品管理**
  - 日用品库存（洗衣液、纸巾、食品）
  - 库存预警（低于阈值提醒）
  - 购物清单生成

#### 5. 环境管理（Context）- 环境维度
- [ ] **家庭环境监测**
  - 温度、湿度、空气质量
  - 光照强度、噪音水平
  - 环境数据历史记录
  - 环境趋势分析
- [ ] **场景管理**
  - 场景定义（回家模式、睡眠模式、离家模式）
  - 场景触发条件（时间/位置/手动）
  - 场景动作设置（Phase3 硬件联动）
- [ ] **空间管理**
  - 房间/区域划分
  - 空间功能定义
  - 空间使用频率统计

#### 6. 数据看板（Dashboard）
- [ ] **家庭概览**
  - 家庭成员数、待办任务数、即将到来的日程
  - 本周/本月活动统计
  - 家庭健康度评分（基于数据完整性和活跃度）
- [ ] **数据可视化**
  - 时间分配饼图
  - 任务完成趋势图
  - 物品分类统计图
  - 环境数据曲线图
- [ ] **智能提醒**
  - 今日待办事项
  - 即将到期的任务
  - 重要日期提醒
  - 物品维护提醒

#### 7. 数据管理
- [ ] **数据导入**
  - 从 Excel/CSV 导入数据
  - 从其他家庭管理应用迁移
  - 数据格式验证
- [ ] **数据导出**
  - 导出为 JSON/CSV/PDF
  - 选择性导出（按模块/时间范围）
  - 数据脱敏选项
- [ ] **数据备份**
  - 自动备份（每日/每周）
  - 手动备份
  - 备份恢复
- [ ] **数据加密**
  - 敏感数据 AES-256 加密
  - 端到端加密（私有部署）
  - 数据传输 HTTPS

### 服务边界与职责划分

#### HamR 管家负责
✅ 五维数据的采集、存储、展示
✅ 家庭日常事务管理
✅ 数据可视化和统计分析
✅ 数据导入/导出/备份
✅ 为 JiaBu 提供结构化数据

#### HamR 管家不负责
❌ 智能决策和建议（由 JiaBu 负责）
❌ 用户身份认证（由 account.hamr.store 负责）
❌ 智能硬件控制（Phase3）
❌ AI 对话交互（Phase2）
❌ 社交分享功能（Phase2）

### 明确不包含
- 不包含：智能决策功能（由 JiaBu 系统负责 - Phase2）
- 不包含：智能硬件控制（Phase3）
- 不包含：AI 对话助手（Phase2）
- 不包含：社交分享（Phase2）
- 不包含：支付功能（产品免费）

## 🛠 技术方案

### 技术栈
- **前端**：React + TypeScript + Ant Design
- **后端**：Node.js + NestJS / Go + Gin
- **数据库**：PostgreSQL（关系数据）+ MongoDB（灵活数据）
- **缓存**：Redis
- **存储**：MinIO（照片、附件）
- **加密**：AES-256（敏感数据）
- **域名**：app.hamr.store
