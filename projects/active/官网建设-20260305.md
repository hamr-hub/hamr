## 🎯 项目概述

### 项目背景
作为 HamR 家庭智能助理平台的门户网站，hamr.store 是用户了解产品、建立信任的第一触点。需要传递"温暖、可信赖、像家一样"的品牌感受，让家庭用户感到安心。

### 项目目标
- 目标1：上线 hamr.store 官网，展示产品理念和核心价值
- 目标2：完成品牌视觉设计，传递温暖家庭氛围
- 目标3：提供产品下载/注册入口，引导用户进入 app.hamr.store
- 目标4：SEO 优化，提升品牌搜索可见性

### 项目价值
- **用户价值**：快速了解产品定位，建立信任感
- **业务价值**：品牌展示窗口，用户获取入口
- **技术价值**：静态站点，高性能、低成本

## 📐 项目范围

### 核心职责
官网是 HamR 平台的**品牌展示门户**，负责：
1. **品牌传达**：传递"温暖、可信赖、像家一样"的产品理念
2. **产品介绍**：清晰展示 HamR 管家和 JiaBu 决策的核心功能
3. **用户导流**：引导用户注册、下载、或进入应用
4. **信任建立**：展示隐私承诺、开源理念、安全保障

## 🛠 技术方案

### 技术栈
- **前端框架**：Vite + React (TypeScript)
- **UI库**：Tailwind CSS + Framer Motion + Lucide Icons
- **托管**：Cloudflare Pages / Vercel
- **CDN**：Cloudflare
- **域名**：hamr.store

## 📝 关键决策记录

### [2026-03-05] 技术栈选择
- **背景**：需要快速上线、高性能、低成本的官网

- **影响范围**：开发效率、运维成本
