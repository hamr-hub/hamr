## 🎯 项目概述

### 项目背景
开发者和用户需要在正式部署前快速体验 HamR 功能，需要提供在线演示环境，降低试用门槛。

### 项目目标
- 目标1：搭建在线演示环境，提供脱敏示例数据
- 目标2：支持用户注册临时账号体验
- 目标3：自动重置数据（防止滥用）
- 目标4：展示核心功能演示

### 项目价值
- **用户价值**：快速体验产品，无需本地部署
- **业务价值**：降低试用门槛，提升转化率
- **技术价值**：验证系统稳定性

## 📐 项目范围

### 核心职责
在线演示环境是 HamR 平台的**产品体验沙箱**，负责：
1. **零门槛体验**：用户无需部署即可快速体验 HamR 核心功能
2. **功能展示**：展示五维管理和数据看板的完整使用流程
3. **开发者调试**：为开发者提供 API 调试的真实环境
4. **环境安全**：自动数据重置、限流保护，防止滥用

### 包含的功能

#### 1. 演示环境部署
- [ ] **完整功能实例**
  - 部署与 app.hamr.store 相同的代码
  - 独立的数据库实例（不共享生产数据）
  - 独立的对象存储实例
  - 独立的 Redis 实例
- [ ] **环境隔离**
  - 与生产环境完全隔离
  - 独立的域名（demo.hamr.top）
  - 独立的 SSL 证书
  - 独立的监控

#### 2. 临时账号管理
- [ ] **账号注册**
  - 邮箱快速注册（简化流程，无需二步验证）
  - 临时 Demo 账号（无需注册直接体验）
  - 预设演示账号（admin@demo.hamr.top / demo123）
- [ ] **账号限制**
  - 单个用户最多创建 1 个家庭
  - 家庭最多 5 个成员
  - 数据存储限制（100MB/账号）
  - 账号有效期（7 天自动过期）
- [ ] **使用限制**
  - 功能水印提示"这是演示环境"
  - 禁用邮件发送（防止垃圾邮件）
  - 禁用 Webhook 回调
  - API 调用限制（10 次/分钟）

#### 3. 示例数据
- [ ] **预填充数据集**
  - 示例家庭（4 口之家：爸爸、妈妈、女儿、儿子）
  - 示例人员关系图谱
  - 示例日历日程（过去 30 天 + 未来 30 天）
  - 示例任务列表（已完成、进行中、待办各若干）
  - 示例物品清单（家电、家具、图书若干）
  - 示例大事记（搬家、生日、旅行记录）
  - 示例环境数据（温度、湿度历史）
- [ ] **数据脱敏**
  - 使用虚构姓名和信息
  - 无真实联系方式
  - 照片使用免费素材
  - 地址使用虚构地点

#### 4. 数据自动重置
- [ ] **重置策略**
  - 每日凌晨 3:00（UTC+8）自动重置
  - 重置前 1 小时在页面弹窗提醒
  - 重置过程中显示维护页面（约 5-10 分钟）
  - 重置后恢复为预设示例数据
- [ ] **重置范围**
  - 清除所有临时注册账号
  - 保留预设演示账号
  - 恢复示例数据到初始状态
  - 清除上传的文件和照片
  - 重置 Redis 缓存

#### 5. 功能引导
- [ ] **交互式引导**
  - 首次登录弹出引导流程（Product Tour）
  - 分步引导重点功能（5-7 步）
  - 引导可跳过、可重新触发
- [ ] **功能标注**
  - 各功能区域添加 Tooltip 说明
  - 示例数据标注"示例"标签
  - 功能限制说明（不可用功能灰色标注）
- [ ] **转化引导**
  - 体验完成后引导注册正式账号
  - 提供"私有部署"入口
  - 提供"了解更多"链接

#### 6. 限流与安全保护
- [ ] **访问限流**
  - IP 限制（单 IP 每分钟最多 30 次请求）
  - 全局并发限制（最多 100 并发用户）
  - 超限返回 429 + 重试等待时间
- [ ] **安全防护**
  - 禁止爬虫（robots.txt + 速率检测）
  - SQL 注入防护
  - XSS 防护
  - 文件上传大小限制（5MB/文件）
  - 文件类型白名单（图片、文档）
- [ ] **监控告警**
  - 异常流量检测
  - 服务健康检查
  - 资源使用率监控

### 服务边界与职责划分

#### 在线演示负责
✅ 提供功能完整的体验环境
✅ 维护示例数据和临时账号
✅ 自动数据重置和环境维护
✅ 功能引导和转化引导

#### 在线演示不负责
❌ 真实数据存储（定期清除）
❌ 长期账号管理（由 account.hamr.store 负责）
❌ 生产级 SLA 保障
❌ 客户支持

### 明确不包含
- 不包含：生产数据（仅示例数据）
- 不包含：长期数据存储（定期清理）
- 不包含：完整功能（部分功能受限）
- 不包含：SLA 保障（尽力而为）

## 🛠 技术方案

### 技术栈
- **应用**：基于 app.hamr.store 代码
- **数据库**：独立的 PostgreSQL（演示数据）
- **定时任务**：Cron（数据重置）
- **限流**：Redis
- **域名**：demo.hamr.top

### 架构设计
```
用户 → demo.hamr.top（演示环境）
     ├─ 临时账号注册
     ├─ 示例数据展示
     ├─ 限流保护
     ↓
   独立数据库（定期重置）
```

### 技术依赖
- app.hamr.store 代码库
- PostgreSQL 独立实例
