## 🎯 项目概述

### 项目背景
HamR 承诺支持私有部署，用户可在自己的服务器上部署 HamR，确保数据完全自主。需要提供详细的部署指南，降低部署门槛。

### 项目目标
- 目标1：提供 Docker Compose 一键部署方案
- 目标2：提供 Kubernetes 生产级部署方案
- 目标3：编写部署故障排查文档
- 目标4：提供部署脚本和配置示例

### 项目价值
- **用户价值**：快速完成私有部署，保证数据自主权
- **业务价值**：兑现隐私承诺，吸引企业用户
- **技术价值**：标准化部署流程

## 📐 项目范围

### 核心职责
部署指南站是 HamR 平台的**私有部署支持中心**，负责：
1. **部署方案提供**：Docker Compose 快速部署 + K8s 生产部署
2. **部署脚本维护**：一键部署脚本的开发和更新
3. **运维指导**：备份恢复、版本升级、故障排查
4. **配置文档**：所有可配置项的详细说明

### 包含的功能

#### 1. Docker Compose 部署指南（快速部署）
- [ ] **环境准备**
  - 系统要求（Linux/macOS/Windows，最低 2核4G）
  - Docker 安装指南（各操作系统）
  - Docker Compose 安装指南
  - 网络要求（开放端口列表）
- [ ] **快速开始**
  - 下载部署包（docker-compose.yml + .env.example）
  - 环境变量配置说明
  - `docker compose up -d` 一键启动
  - 首次访问与初始化设置
  - 验证部署成功
- [ ] **服务组件说明**
  - 服务列表（Web 前端、API 服务、PostgreSQL、Redis、MinIO）
  - 端口映射表
  - 数据卷挂载说明
  - 资源限制配置
- [ ] **自定义配置**
  - 域名配置（Nginx 反向代理）
  - SSL 证书配置（Let's Encrypt 自动申请）
  - 邮件服务配置（SMTP）
  - 存储配置（本地/MinIO/S3）
  - 数据库连接配置

#### 2. Kubernetes 部署指南（生产环境）
- [ ] **K8s 环境准备**
  - 集群要求（至少 3 节点，各 4核8G）
  - kubectl 安装与配置
  - Helm 安装
  - 存储类（StorageClass）配置
- [ ] **Helm Chart 部署**
  - 添加 HamR Helm 仓库
  - values.yaml 配置说明
  - `helm install` 部署命令
  - 部署验证
- [ ] **生产级配置**
  - 高可用配置（多副本）
  - 资源限制（requests/limits）
  - 自动扩缩容（HPA）
  - 持久化存储（PVC）
  - Ingress 配置
  - TLS 证书管理（cert-manager）
- [ ] **运维操作**
  - 滚动更新
  - 回滚操作
  - 日志查看
  - Pod 调试

#### 3. 配置参数详解
- [ ] **环境变量清单**
  ```bash
  # 数据库配置
  DB_HOST=localhost
  DB_PORT=5432
  DB_NAME=hamr
  DB_USER=hamr
  DB_PASSWORD=your_secure_password

  # Redis 配置
  REDIS_HOST=localhost
  REDIS_PORT=6379
  REDIS_PASSWORD=your_redis_password

  # JWT 配置
  JWT_SECRET=your_jwt_secret
  JWT_ACCESS_EXPIRES=15m
  JWT_REFRESH_EXPIRES=7d

  # 邮件配置
  SMTP_HOST=smtp.example.com
  SMTP_PORT=587
  SMTP_USER=noreply@hamr.store
  SMTP_PASSWORD=your_smtp_password

  # 存储配置
  STORAGE_TYPE=minio  # local / minio / s3
  MINIO_ENDPOINT=localhost:9000
  MINIO_ACCESS_KEY=your_access_key
  MINIO_SECRET_KEY=your_secret_key
  MINIO_BUCKET=hamr-files

  # 应用配置
  APP_URL=https://app.yourdomain.com
  ACCOUNT_URL=https://account.yourdomain.com
  ```
- [ ] **配置分组说明**
  - 必填配置 vs 可选配置
  - 安全敏感配置（密码、密钥）
  - 性能调优配置
  - 功能开关配置

#### 4. 域名与 SSL 配置
- [ ] **域名规划建议**
  - 单域名方案（使用路径区分服务）
  - 多子域名方案（推荐）
  - 域名 DNS 配置
- [ ] **SSL 证书**

  - 证书自动续期

#### 5. 数据备份与恢复
- [ ] **备份策略**

- [ ] **恢复操作**

  - 恢复验证清单

#### 6. 版本升级指南
- [ ] **升级流程**
  - 查看当前版本

  - 备份当前数据
  - 拉取新版本镜像
  - 执行数据库迁移
  - 重启服务
  - 验证升级结果
- [ ] **回滚方案**
  - 回退到上一版本镜像
  - 恢复数据库备份
  - 验证回滚结果

- [ ] **日志分析**

  - 存储服务检查
  - 网络连通性检查
