## 🎯 项目概述

### 项目背景
JiaBu（家布）是 HamR 平台的智能决策助手，基于 HamR 管家收集的家庭数据，提供智能决策建议、幸福感评估、未来规划模拟等功能，是产品的核心价值升华。

### 项目目标
- 目标1：实现家庭幸福感评估模型
- 目标2：提供智能提醒和决策建议
- 目标3：支持事物发展模拟（场景预测）
- 目标4：生成家庭幸福报告
- 目标5：环境监测与行动优化

### 项目价值
- **用户价值**：从数据管理升级到智能决策，提升家庭幸福感
- **业务价值**：差异化竞争力，AI 赋能家庭管理
- **技术价值**：AI 模型应用、数据分析实践

## 📐 项目范围

### 核心职责
JiaBu 是 HamR 平台的**智能决策引擎**，负责：
1. **数据分析**：对 HamR 管家收集的五维数据进行深度分析
2. **幸福感评估**：量化家庭幸福指数，识别改善方向
3. **智能建议**：基于数据分析提供个性化决策建议
4. **场景模拟**：预测不同决策的潜在影响
5. **报告生成**：定期生成家庭幸福报告

### 包含的功能

#### 1. 幸福感评估模型
- [ ] **五维幸福指数**
  - **人际维度**（People Score）
    - 家庭互动频率（共同活动、交流频次）
    - 关系亲密度变化趋势
    - 重要日期遗忘率
    - 成员间冲突频率（基于事务记录）
  - **时间维度**（Time Score）
    - 家庭时间占比（vs 工作/个人时间）
    - 日程冲突率
    - 计划完成率
    - 时间分配均衡度
  - **事务维度**（Event Score）
    - 任务完成率和及时性
    - 家庭活动频率和多样性
    - 大事记正面/负面比率
    - 决策执行效果
  - **物品维度**（Thing Score）
    - 资产管理完善度
    - 物品使用效率
    - 消耗品管理及时性
    - 家庭空间利用率
  - **环境维度**（Context Score）
    - 居住环境舒适度
    - 环境改善趋势
    - 场景利用率
- [ ] **综合幸福指数**
  - 五维加权综合评分（0-100 分）
  - 权重可自定义（默认各 20%）
  - 趋势对比（周/月/季/年）
  - 同类家庭对比（匿名基准数据）
- [ ] **幸福感预警**
  - 指数持续下降告警
  - 单维度异常检测
  - 改善建议推送

#### 2. 智能提醒系统
- [ ] **基于规则的提醒**
  - 重要日期提醒（生日、纪念日、节假日）
  - 任务截止提醒
  - 物品维护提醒（保养、更换）
  - 消耗品补货提醒
- [ ] **基于分析的提醒**
  - "您已经 X 天没有全家一起活动了"
  - "本月家庭时间比上月减少了 30%"
  - "有 X 个过期任务未处理"
  - "XX 的生日即将到来，往年您通常提前 X 天准备"
- [ ] **个性化建议**
  - 基于历史偏好推荐活动
  - 基于季节/天气推荐户外活动
  - 基于家庭成员年龄推荐亲子活动
  - 基于预算推荐经济实惠的家庭活动

#### 3. 决策辅助引擎
- [ ] **场景模拟**
  - 输入决策选项（如：搬家到 A 地 vs B 地）
  - 分析影响因素（通勤时间、学区、环境、成本）
  - 生成多维度对比报告
  - 历史类似决策参考
- [ ] **风险评估**
  - 决策风险量化（财务风险、时间风险、关系风险）
  - 最佳/最差/中位情景分析
  - 风险缓解建议
- [ ] **决策推荐**
  - 基于家庭优先级排序推荐
  - 利弊分析（Pros & Cons）
  - 决策时间线（何时执行最佳）
- [ ] **决策跟踪**
  - 决策执行进度
  - 决策结果评估（事后复盘）
  - 决策经验积累

#### 4. 幸福报告生成
- [ ] **周报**
  - 本周幸福指数变化
  - 关键事件回顾
  - 下周建议
- [ ] **月报**
  - 月度幸福趋势
  - 五维分析对比
  - 改善重点建议
  - 家庭亮点事件
- [ ] **年报**
  - 年度幸福回顾
  - 重大事件时间线
  - 成长与变化总结
  - 新年展望和建议
- [ ] **报告可视化**
  - 雷达图（五维评分）
  - 趋势折线图
  - 热力图（活动频率）
  - 对比柱状图

#### 5. 环境监测与优化
- [ ] **环境分析**
  - 家庭环境数据趋势分析
  - 异常环境检测（温度骤变、湿度异常）
  - 环境与健康关联分析
- [ ] **优化建议**
  - 根据天气预报建议开窗通风
  - 根据空气质量建议使用净化器
  - 根据季节建议调整室内温度
  - 根据家庭成员作息建议灯光调节

### 算法设计（MVP - 规则引擎）

#### 规则引擎架构
```
数据输入（五维数据）
  ↓
预处理（清洗、归一化）
  ↓
规则匹配引擎
├─ 触发规则（if-then）
├─ 评分规则（加权求和）
├─ 建议规则（条件→建议）
└─ 告警规则（阈值判断）
  ↓
结果输出（评分、建议、报告）
```

#### 幸福指数计算公式（MVP）
```
总分 = Σ(维度权重 × 维度得分)

维度得分 = Σ(指标权重 × 指标标准化值)

示例：
人际维度 = 0.3×互动频率 + 0.3×亲密度趋势 + 0.2×日期记忆率 + 0.2×冲突低频率
时间维度 = 0.3×家庭时间占比 + 0.25×计划完成率 + 0.25×均衡度 + 0.2×冲突低率
...
```

#### Phase3 机器学习演进路线
