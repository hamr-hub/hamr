# 已归档项目

## 📂 archived/ - 已完成/已取消的项目

存放已结束的项目文档，便于历史回顾和经验总结。

### 归档规则

**何时归档**：
- 项目已完成并上线
- 项目已取消
- 项目已暂停超过 3 个月

**归档操作**：
```bash
# 从 active/ 移动到 archived/
mv ../active/项目名称-20260304.md ./
```

### 状态标记（可选）

可在文件名添加状态后缀：
- `项目名称-20260304-completed.md` - 已完成
- `项目名称-20260304-cancelled.md` - 已取消
- `项目名称-20260304-paused.md` - 已暂停

### 定期清理

建议每季度回顾归档文档：
- 删除价值不大的文档
- 提取可复用的经验到 `resources/`
- 保留有参考价值的项目文档
