# 项目背景与上下文

## 项目信息

- **项目名称**：HAMR - 项目规划知识库
- **项目用途**：为 HamR（家庭智能助理）项目提供集中化的项目管理与文档存储
- **主要语言**：中文
- **文档格式**：Markdown + JSON

## 核心理念

### 1. 结构化存储
项目文档按照明确的目录结构组织：
- `projects/` - 项目文档（活跃/归档）
- `planning/` - 规划文档（路线图/提案/评审）
- `execution/` - 执行文档（迭代/回顾/报告）
- `templates/` - 文档模板
- `resources/` - 参考资料

### 2. 双索引系统
- **Markdown 文档**：人类可读，详细描述
- **JSON 注册表**：机器可读，快速索引

### 3. AI 辅助优先
- 文档结构支持 AI 快速定位
- 元数据标准化便于 AI 理解
- 模板化减少重复劳动

## 项目目标

1. **信息集中化**：避免项目信息散落在各处
2. **过程可追溯**：通过 Git 追踪项目演进
3. **知识可沉淀**：从项目中提炼可复用经验
4. **协作可扩展**：支持团队成员并行工作

## 技术约束

- **无代码依赖**：纯文档项目，不涉及编程语言
- **Git 版本控制**：所有变更必须提交到 Git
- **人机协作**：人类编写核心内容，AI 辅助格式化和检索
- **离线可用**：不依赖外部服务，本地即可完整使用
