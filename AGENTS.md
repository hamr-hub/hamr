# AGENTS.md

项目规划知识库的 AI 助手工作指南。

## WHY: 目标与用途

这是一个项目管理知识库，用于存储项目规划、执行和复盘文档。核心价值是提供结构化的文档组织方式，支持项目生命周期管理。

## WHAT: 技术栈

- 格式：Markdown 文档
- 版本管理：Git
- 编辑器：支持任意 Markdown 编辑器
- 语言：中文

## HOW: 核心工作流

```bash
# 查看目录结构
tree -L 2

# 提交变更
git add .
git commit -m "描述变更内容"
```
